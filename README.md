# Food Ordering System (Java)

## 📌 Overview

This is a console-based Food Ordering System developed using Java. It allows users to select food items, add them to an order, and calculate the total bill.

---

## 🚀 Features

* Menu-driven console application
* Supports Veg, Non-Veg, and Dessert items
* Uses abstraction and inheritance
* Dynamic price calculation
* Displays complete order with total bill
* Simple and easy-to-use interface

---

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Inheritance & Abstraction
* Collections (ArrayList)
* Scanner for input

---

## 📂 Project Structure

* `FoodItem` → Abstract parent class
* `VegItem`, `NonVegItem`, `Dessert` → Child classes
* `Order` → Handles item list and billing
* `Fulllength2.java` → Main class with menu

---

## ▶️ How to Run

1. Open Command Prompt / Terminal
2. Compile the program:

   ```bash
   javac Fulllength2.java
   ```
3. Run the program:

   ```bash
   java Fulllength2
   ```

---

## 📊 How It Works

* User selects food category from menu
* Items are added to the order
* Each item calculates price based on type
* User can view full order and total bill
* Program continues until user exits
